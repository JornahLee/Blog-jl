# blog-jl
个人博客

开发此博客的目的：
1. 便于自己整合收集笔记，问题查阅，积累
2. 巩固复习后端框架与知识点，避免遗忘
3. 涉猎前端开发，且能根据自我习惯定制化开发前后端页面


## 技术栈：
前端：vue 2.x, 地址：https://github.com/JornahLee/blog-jl-front

后端: SpringBoot Redis MySQL Mybatis-plus

全文检索：Elasticsearch

分布式配置中心 Nacos

链路追踪 Jaeger

jwt

持续集成/发布： Jenkins Docker Nginx
